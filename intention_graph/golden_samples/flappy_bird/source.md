# Flappy Bird Golden Sample

**Product**: Flappy Bird (Dong Nguyen, 2013)
**Type**: Game (arcade)
**Platform**: iOS/Android (portrait, 1080x1920)

## Source
- Original game design knowledge (widely documented)
- Screen layout based on original game screenshots
- 4 screens: Start, Gameplay, Game Over, Leaderboard

## Screens
1. **Start Screen**: Title logo, bird preview, Play + Score buttons
2. **Gameplay**: Sky background, pipes, bird, score counter (top center), tap area
3. **Game Over**: Game Over text, score panel with medal, Play + Score buttons
4. **Leaderboard**: Popup with score list, OK button to close

# 植物大战僵尸 (Plants vs Zombies) Golden Sample

## Source
Created from the classic PvZ game design (PopCap Games, 2009).
Based on publicly known game mechanics and UI layout.

## Screens (6)
1. **main_menu** — Title screen with adventure mode and mini-game buttons
2. **level_select** — Map-based level progression with node-style layout
3. **plant_select** (popup) — Card grid for choosing up to 6 plants before battle
4. **gameplay** — 5-lane grid with plant bar, sun counter, zombie waves
5. **pause_popup** — Continue/restart/quit options
6. **game_over** — "Zombies ate your brains!" with retry and menu buttons

## Complexity
- Type: casual
- 6 screens (4 pages + 2 popups)
- 43 total elements
- Key navigation: menu → level select → plant select → gameplay → game over

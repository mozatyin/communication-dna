# 俄罗斯方块 (Tetris) Golden Sample

## Source
Based on the classic Tetris game design (Alexey Pajitnov, 1984).
Reference: standard mobile Tetris implementations.

## Screens (4)
1. **main_menu** — Title with neon-style TETRIS text, start and leaderboard buttons
2. **gameplay** — 10x20 grid, next block preview, score/level/lines display
3. **game_over** — Final score, lines cleared, level reached, retry/menu buttons
4. **leaderboard** (popup) — Top 5 high scores

## Complexity
- Type: arcade
- 4 screens (3 pages + 1 popup)
- 30 total elements
- Key navigation: menu → gameplay → game_over → menu/leaderboard

# 贪吃蛇 (Snake) Golden Sample

## Source
Based on the classic Snake game (Nokia, 1998).
Reference: modern mobile Snake implementations.

## Screens (3)
1. **main_menu** — Title, high score display, start button
2. **gameplay** — Grid area, snake, food, score, d-pad controls
3. **game_over** — Final score, retry and menu buttons

## Complexity
- Type: arcade
- 3 screens (3 pages, no popups)
- 20 total elements
- Key navigation: menu → gameplay → game_over → menu/gameplay
- Simplest golden sample — no popup, minimal screens
